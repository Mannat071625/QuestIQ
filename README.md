# QuestIQ
An interactive web-based quiz game wherein players solve riddles and clues across multiple levels with a timer and score system.

---

## 🚀 Features
- 🧩 Multiple puzzle levels
- ⏱ Timer-based challenges
- ❤️ Lives system
- 🎯 Score tracking
- 🎨 Clean and responsive UI

---

## 🛠 Tech Stack
- HTML
- CSS
- JavaScript
